<h2>Explorer TXT Editor</h2>
<p>Explorer TXT Editor是一个极其简单的TXT文件编辑器，基于.NET Framework的Windows窗体应用，<strong>完全使用DeepSeek-V3.2制作</strong>。</p>
<p>目前我还没有测试此应用能否编辑其他格式的文件，因此，<strong>请勿将其作为notepad.exe的替代品</strong>。</p>
<p><strong>本程序目前没有可用安装包，请自行使用Visual Studio 2022编译!</strong></p>
<p>Explorer TXT Editor is an extremely simple TXT file editor. A Windows Forms application based on the .NET Framework, and is <strong>entirely built using DeepSeek-V3.2.</strong></p>
<p>I haven't tested this app for editing files in other formats yet. So <strong>DO NOT use it as a replacement for Notepad.exe</strong>.</p>
<p><strong>There are currently no available installation packages in this program. Please use Visual Studio 2022 to compile by yourself!</strong></p>
